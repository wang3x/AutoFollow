# 强效跟随 AutoFollow

> FF14 卫月 (Dalamud) 自动跟随插件 — **vnavmesh 寻路跟随 · 自动疾跑 · 循环插件协同**

---

## 安装

### 方式一：通过 Release 下载

1. 从 [Releases](https://github.com/wang3x/AutoFollow/releases) 下载最新版 `AutoFollow_v*.zip`
2. 解压到 `%APPDATA%\XIVLauncher\devPlugins\AutoFollow\` 目录
3. 游戏中 `/xlload` 或重启游戏
4. 插件管理器 → 开发工具 → 已安装的开发插件 → 启用

### 方式二：本地插件

1. 下载 zip 解压
2. 卫月设置 → 插件 → 本地插件 → 添加 DLL → 选择 `AutoFollow.dll`

### ⚠️ 依赖

| 插件 | 必需 | 安装地址 |
|------|------|---------|
| **vnavmesh** | ✅ **必须** | 第三方仓库 `https://puni.sh/api/repository/veyn` |
| BossMod / BossModReborn | 可选 | 自动跟随暂停后接管战斗 |
| RotationSolverReborn | 可选 | 自动输出，跟随期间自动暂停 |

## 快速开始

```
/ftar          → 跟随当前选中的玩家
/ft <玩家名>    → 按名字跟随，如 /ft 小明
/ff            → 切换跟随开/关
```

选好目标后插件自动通过 vnavmesh 寻路走到目标身边。跟随期间会自动处理疾跑、战斗暂停、循环插件协同。

---

## 所有命令

| 命令 | 动作 | 说明 |
|------|------|------|
| `/ff` | ToggleFollow | 切换跟随开/关 |
| `/ftar` | FollowCurrentTarget | 跟随当前选中的目标 |
| `/ft <名>` | SetFollowTarget | 按名字设置跟随目标 |
| `/fes` | EmergencyStop | 紧急停止（停止 + 暂停循环） |
| `/flp` | PauseLoop | 暂停循环插件 |
| `/flr` | ResumeLoop | 恢复循环插件 |
| `/flt` | ToggleLoop | 切换循环插件 |
| `/fst` | StatusReport | 输出状态报告到聊天 |
| `/fdbg` | ToggleDebugWindow | 打开/关闭主窗口 |

支持自定义命令：打开主窗口 → 设置页 → 自定义命令面板可增删改。

## 跟随机制

### 距离状态机

每 1 秒扫描一次目标坐标（间隔可调），检测到移动超过 0.5 码即发送新路径给 vnavmesh。

```
进入战斗区 (默认 ≤10码)  → 暂停跟随，恢复循环插件 → BossMod 接管
离开战斗区 (默认 >30码)  → 恢复跟随，暂停循环插件
脱战后 >1秒            → 恢复跟随（距目标≤10码时不恢复，防反复横跳）
```

- 设置目标后有 **2 秒启动保护期**，防止一出门就触发战斗暂停
- **Boss 监测**：扫描全场景 BattleNpc，分级判断（Lv≥80 HP>20倍 / Lv50-79 HP>15倍 / Lv<50 HP>10倍），Boss 战中不恢复跟随

### 自动疾跑

疾跑和坐骑二选一互斥，通过设置页切换。

| 模式 | 行为 |
|------|------|
| 无脑疾跑 | 一直开着疾跑 |
| 智能疾跑 | 目标>20码或目标正在疾跑时自动开 |
| 仅战斗疾跑 | 仅在战斗状态中开疾跑 |
| 坐骑模式 | 非战斗自动上坐骑，不开疾跑 |

### 智能跟随按钮

主窗口/迷你窗口的 **跟随** 按钮：

- 选中 **玩家** → 直接跟随该玩家
- 选中 **敌方 NPC** → 跟随该 NPC 的当前目标（通常是坦克）
- 其他情况 → 提示无法确定目标

### 紧急停止

默认热键 **F8** 按一次即触发。可在设置页改绑其他按键。

触发后：停止跟随 + 暂停循环插件。需手动 `/ff` 恢复。

### 地图黑名单

设置页可添加地图 ID。在黑名单地图中自动暂停跟随。

## 主窗口

游戏内 `/fdbg` 或 `/fcfg` 打开。

### 标签页

| 页签 | 功能 |
|------|------|
| **设置** | 距离阈值、疾跑/坐骑、热键绑定、循环插件命令、自定义命令 |
| **坐标** | 玩家/目标坐标、移动到旗、飞到旗、手动输入坐标移动 |
| **插件状态** | 检测 vnavmesh / BossMod / RotationSolver 是否已连接 |
| **使用说明** | 本说明的游戏内版本 |
| **地图黑名单** | 添加/删除黑名单地图 ID |
| **指令日志** | 颜色分类的操作日志，支持过滤和复制 |

### 迷你窗口

默认打开，显示：
- **指示圈**：绿=跟随中、黄=暂停、红=紧急停止
- **目标名 + 距离**
- **急停/恢复** 按钮
- **跟随** 按钮（智能跟随）

## 循环插件协同

在设置页填写循环插件的暂停/恢复命令（默认为 RotationSolverReborn 的 `/rotation off` 和 `/rotation Auto`）。

**工作流程：**

```
跟随中目标走远     → 暂停循环插件
                   → vnavmesh 寻路绕障碍追过去

接近目标进入战斗区 → 暂停跟随，恢复循环插件
                   → BossMod 接管战斗移动+输出

脱战              → 恢复跟随，暂停循环插件
                   → 继续追目标
```

## 常见问题

**Q: 插件加载不出来？**
A: 检查 `AutoFollow.json` 中 `DalamudApiLevel` 是否匹配卫月版本（国服 API 15）。检查 `ImGui.NET.dll` 是否和 `AutoFollow.dll` 在同一目录。`/xllog` 查看错误日志。

**Q: 跟了不走？**
A: 检查 vnavmesh 是否已安装启用、NavMesh 是否已加载（vnavmesh 窗口显示绿色）。查看主窗口指令日志是否有 move 记录。尝试坐标页手动输入坐标移动。

**Q: 循环插件不自动暂停/恢复？**
A: 检查设置页的循环插件命令是否正确。默认是 RotationSolverReborn 的 `/rotation off` / `/rotation Auto`。其他插件需要自己改。

**Q: 怎么换跟随目标？**
A: `/ft <新玩家名>` 即可切换，或 `/ft` 不加参数清除目标。也可以用 `/ftar` 重新选一个。

**Q: 紧急停止没反应？**
A: 检查设置页热键是否被绑定，是否被其他插件占用。当前是按下即触发（无长按）。
