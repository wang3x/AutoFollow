# 强效跟随 AutoFollow

基于 **vnavmesh 寻路**的 FF14 自动跟随插件。设定目标 → 自动走到身边 → 战斗时暂停 → 脱战继续追。

---

## 安装

1. **装 vnavmesh**（必需）：卫月 → 第三方仓库 → 添加 `https://puni.sh/api/repository/veyn` → 安装
2. **装本插件**：将 `成品` 文件夹里的 `AutoFollow.dll` + `AutoFollow.json` + `ImGui.NET.dll` 放到任意目录，卫月 → 插件 → 本地插件 → 添加 DLL

---

## 快速上手

```
/ftar         → 跟随当前选中的玩家
/ff           → 切换跟随开/关
F8            → 紧急停止
/fdbg         → 打开设置窗口
```

**迷你窗口**（默认显示在屏幕上）：
- **▶ 跟随** — 点击开始跟随当前选中目标
- **⏸ 暂停** — 跟随中点一下急停
- **▶ 继续** — 暂停后恢复跟随旧目标
- **右键**迷你窗口 → 跟随队伍成员 / 状态报告 / 关闭窗口
- `/fdbg` 重新打开被关闭的迷你窗口

---

## 插件自动做的事

| 场景 | 行为 |
|------|------|
| 跟人走远 | 暂停循环插件 → vnavmesh 寻路追过去 |
| 接近目标 | 恢复循环插件 → BossMod 接管战斗 |
| 脱战 | 暂停循环插件 → 继续跟人 |
| 目标距离 > 30y | 自动开疾跑跟上 |
| 战斗中 | 自动暂停跟随，打完恢复 |
| 目标丢失 | 自动暂停，重新选中再继续 |

## 其他命令

| 命令 | 作用 |
|------|------|
| `/ft <玩家名>` | 按名字跟随 |
| `/fes` | 紧急停止 |
| `/flp` / `/flr` / `/flt` | 暂停/恢复/切换循环插件 |
| `/fst` | 查看状态报告 |

## 设置项一览

| 参数 | 默认值 | 说明 |
|------|--------|------|
| 进入战斗区 | 10y | 距目标 ≤ 此值时暂停跟随 |
| 离开战斗区 | 30y | 距目标 > 此值时恢复跟随 |
| 疾跑触发 | 20y | 目标超过此距离自动开疾跑 |
| 扫描间隔 | 2 秒 | 每隔多久检测目标坐标 |
| 移动阈值 | 5y | 目标移动超过此距离才发送新路径 |
| 紧急停止热键 | F8 | 随时按，立刻停 |
| 聊天提示 | 关闭 | 打开后跟随状态会发到聊天框 |

> 全部设置在游戏内 `/fdbg` → **设置** 页调整

## 常见问题

**跟了不走？** 检查 vnavmesh 是否启用、NavMesh 是否加载。查看 `/fdbg` → 指令日志看是否有 move 记录。

**循环插件不自动停？** 默认支持 RotationSolverReborn。用 AEAssist 则在设置页填暂停命令 `/aestop`，恢复留空。

**怎么换目标？** `/ft <新名字>` 或 `/ft` 清除目标后重新 `/ftar`。

**迷你窗口不见了？** 右下角卫月图标 → 插件列表 → 强效跟随 → 显示迷你窗口。
